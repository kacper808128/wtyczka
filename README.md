# Recruitment Form Filler Chrome Extension

This Chrome extension helps you automatically fill out recruitment forms.

## How to install

1.  Open Chrome and go to `chrome://extensions/`.
2.  Enable "Developer mode" in the top right corner.
3.  Click on "Load unpacked".
4.  Select the directory where you have saved the extension files.

## How to use

1.  Navigate to a web page with a recruitment form.
2.  Click on the extension icon in the Chrome toolbar.
3.  Click the "Fill Form" button in the popup.
4.  The extension will automatically fill in the form fields based on the data in `data.json`.

## How to customize

1.  Open the `data.json` file in a text editor.
2.  Replace the dummy data with your own information.
3.  Save the file and reload the extension in `chrome://extensions/`.
